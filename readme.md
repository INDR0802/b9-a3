# Hockey Website

Welcome to the Hockey website! This website is dedicated to providing information and updates about hockey, created using HTML and vanilla CSS.

## Live Demo

You can view the live website [here](https://indr0802.github.io/b9-a3/).

## Features

- **Clean Design**: The website features a sleek and professional design, focusing on providing an excellent user experience.
- **Responsive Layout**: The layout is responsive and adapts to different screen sizes, ensuring a great experience on both desktop and mobile devices.
- **Team Information**: Detailed information about various hockey teams.
- **Player Stats**: Up-to-date player statistics and performance metrics.
- **Match Schedules**: A schedule of upcoming matches and events.
- **Latest News**: The latest news and updates from the world of hockey.

## Technologies Used

- **HTML**: For the structure of the website.
- **CSS**: For styling and layout.

## Usage

To use or modify this project:

1. Clone the repository:

   ```sh
   git clone https://github.com/Indr0802/b9-a3.git
   ```

2. Navigate to the project directory:

   ```sh
   cd b9-a3
   ```

3. Open the `index.html` file in your preferred web browser to view the website locally.
